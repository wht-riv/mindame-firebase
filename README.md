# みんだめ v2

言い訳の事前予測と、100％だけを理想にしないフィードバックを用いた習慣支援Webアプリです。

## v2の主な変更

- 固定の「最低ライン」を廃止し、毎回「今日の最初の一歩」を設定
- 1.0 / 0.75 / 0.5 の曖昧な点数制度を廃止
- 結果を「実行・事前休息・未実行・状況変更・回答しない」で記録
- 3日間チャージを廃止し、同じ言い訳の連続使用時は再確認
- 自由記述チケットをいつでも追加可能
- 50～70％メーターを「最近のペース」として維持
- メーターは直近7回の有効な振り返りから計算し、行動・休息・未実行の内訳も表示
- 連続ログイン日数や欠けた日数を利用者に表示しない
- 「みんなの言い訳」は任意・匿名のお楽しみ／共感機能

詳しい更新方法は `UPDATE_GUIDE.md` を参照してください。
